About

The goal of this project is to allow users to better visualize the contemporaries of historical figures.

It is based on the idea that people are often influenced by their contemporaries, and that understanding these relationships can help us better understand history.

Future Work

Planned improvements include adding more detailed biographical data, more advanced search features, and a more interactive timeline visualization.

Acknowledgments

The data used by this project came from Morgane Laouenan et al. ↗. The data was sorted based on a person's notability ranking. Only the top 250,000 historical figures are viewable.

The timeline visualization was created using Timeline by Knightlab ↗.

Original data was provided by Wikipedia ↗ and Wikidata ↗.
